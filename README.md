# Jester - Entertainment Resource Directory

An Electron desktop application that provides a curated directory of entertainment resources including memes, comedy clubs, jokes, and local theaters.

## Features

- **Memes & Humor**: Access popular meme sites like Reddit, 9GAG, Imgur, and meme generators
- **Comedy Clubs**: Find comedy venues in major US cities
- **Jokes**: Get random jokes and access joke APIs/resources
- **Local Entertainment**: Discover theaters, concerts, and events near you
- **Dark/Light Theme**: Toggle between themes for comfortable viewing
- **External Links**: All links open in your default browser for security
- **Search**: Filter resources within each category

## Installation

```bash
# Clone the repository
git clone https://github.com/guildmasterdev/Jester.git

# Navigate to project directory
cd Jester

# Install dependencies
npm install

# Run the application
npm start
```

## Development

```bash
# Run in development mode
npm run dev

# Build for distribution
npm run build
```

## Building for Distribution

The app can be packaged for different platforms:

```bash
# Build for current platform
npm run dist

# Platform-specific builds are configured in package.json
```
